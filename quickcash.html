QuickCash

A cutting-edge Android app for job postings, built in Java with Firebase integration.

Project Details

  • Platform: Android
  • Language: Java
  • Database: Firebase
  • Technologies:
    • Push Notifications
    • Google Maps Integration
    • PayPal Payment Gateway
    • Firebase Authentication and Database
  • Features:
    • Create job posts
    • Apply for jobs
    • User authentication
    • Location-based job suggestions
    • Secure payments using PayPal
